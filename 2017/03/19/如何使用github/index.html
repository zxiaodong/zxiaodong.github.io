如何使用github

安装hexo

  1. 安装node

  2. 安装Git

  3. 打开淘宝镜像http://npm.taobao.org/

  4. 复制npm install -g cnpm –registry=https://registry.npm.taobao.org

  5. 键盘输入windows+r 输入cmd

  6. 输入cnpm install –g hexo-cli

  7. Cd /d D:MD

  8. hexo init hexo

  9. cd hexo

  10. hexo server

  11. 完成创建如图：

  1. 配置完成，打开浏览器，输入http://localhost:4000/

  2. 打开默认hexo页面如下图所示：

  3. 部署到github pages
